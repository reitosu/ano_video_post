本利用規約（以下「本規約」）は、Daisuke106（以下「運営者」）が提供する匿名型動画投稿プラットフォーム **ano_video_post**（以下「本サービス」）の利用に関する条件を定めるものです。

本サービスを利用した時点で、利用者は本規約のすべての条項に同意したものとみなします。同意いただけない場合は、本サービスの利用を直ちに中止してください。

本規約において使用する用語の定義は以下のとおりです。

| 用語 | 定義 |
|------|------|
| 利用者 | 本サービスにアクセスし、利用するすべての者 |
| 投稿コンテンツ | 利用者が本サービスを通じてアップロードした動画・画像・テキスト等 |
| NFT | ブロックチェーン上に記録された非代替性トークン |
| ウォレット | 暗号資産・NFT を管理するための秘密鍵を含む電子財布 |
| 運営者 | 本サービスの開発・運営を行う Daisuke106 |

## 第3条（サービスの概要）

本サービスは以下の機能を提供します。

1. **匿名動画投稿**: アカウント名はユーザー自身が任意に設定できます。実名の開示は不要です
2. **タグ管理**: 動画へのタグ付けおよびタグを基にした検索
3. **NFT 化**: 投稿した動画コンテンツを Ethereum 互換ブロックチェーン上で NFT として発行
4. **Cloudinary 配信**: アップロードされた動画はクラウドストレージ経由で配信されます
